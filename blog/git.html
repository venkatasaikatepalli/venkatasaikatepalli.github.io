Commands to maintain versions of code in GitHub, Gitlab

Getting Started

CommandsDescriptionExample
git config --listlist the git configuration of your accountgit config --list
git config --global user.email "[email]"
Add email for your git commit transactions

git config --global user.email "jhon@sme.com"

git config --global user.name "[name]"
Add name for your git commit transactions

git config --global user.name "Jhon smith"

Initiating & Cloning Project

CommandsDescriptionExample
git initInitialize a local Git repositorygit init
git clone ssh://git@github.com/[username]/[repository-name].gitCreate a local copy of a remote repository

HTTP: git clone https://github.com/john/test.git

SSH: git clone ssh://git@github.com/john/test.git
